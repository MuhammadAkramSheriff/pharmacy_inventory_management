Pharmacy Management System

Language & Framework:
Built using Laravel, a PHP framework known for its robustness and MVC architecture, ensuring efficient development and scalability.

Application Overview:
Automates inventory management and customer records for a pharmacy. Features role-based access control for Owner, Manager, and Cashier.

Login Mechanism:
Utilizes token-based authentication, specifically Laravel Sanctum, to implement user roles. Each user's access is determined by their role, ensuring secure and controlled system access.
